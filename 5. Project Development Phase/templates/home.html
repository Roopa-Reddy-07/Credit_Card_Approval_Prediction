Credit Card Approval Prediction System

Welcome to the Credit Card Approval Prediction System developed as part of the APSCHE SmartBridge Virtual Internship Program.

This application uses Machine Learning algorithms to analyze applicant information and predict whether a credit card application is likely to be Approved or Rejected.

The system has been developed using Python, Flask, Scikit-learn, Random Forest, XGBoost, and other Machine Learning techniques to assist financial institutions in making faster and more accurate credit card approval decisions.
